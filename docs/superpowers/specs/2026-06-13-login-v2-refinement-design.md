# 登录页 v2 优化设计

## 概述

在 v1 基础上三个调整：垂直居中布局、背景图虚化、登录卡片发光。

## 变更

### 1. 布局：垂直居中

```
┌──────────────────────────────┐
│                              │
│       实习能量站               │
│   AI 驱动的实习生成长导航系统    │
│                              │
│   ┌──────────────────────┐   │
│   │  glass card + glow   │   │
│   │     登录表单           │   │
│   └──────────────────────┘   │
│                              │
└──────────────────────────────┘
```

- 外层 `flex-direction: column`，`align-items: center`，`justify-content: center`
- 品牌文字和卡片垂直排列，整体页面居中

### 2. 背景图虚化

- 使用 `::before` 伪元素单独承载 cover.png
- 对伪元素应用 `filter: blur(8px)` + `transform: scale(1.05)`（消除模糊边缘白边）
- 紫色渐变叠加层保持在 `::before` 之上

### 3. 登录卡片发光

- 添加多层 `box-shadow`：
  - `0 0 24px rgba(255,255,255,0.15)` — 白光
  - `0 0 48px rgba(147,112,219,0.25)` — 紫色光晕
  - `0 8px 32px rgba(0,0,0,0.2)` — 底部阴影
- 边框加亮：`rgba(255,255,255,0.3)`（原 0.2）

## 文件变更

| 文件 | 变更 |
|------|------|
| `frontend/src/pages/Login.tsx` | 布局改垂直，卡片加 glow box-shadow |
| `frontend/src/index.css` | 新增 blur 伪元素样式 |

## 技术细节

- 模糊伪元素需要绝对定位，z-index 低于内容
- `scale(1.05)` 补偿 blur 导致的边缘透明，裁切由父容器 `overflow: hidden` 处理
- 其余逻辑（表单、API 调用、角色路由）不变
