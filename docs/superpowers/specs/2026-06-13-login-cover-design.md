# 登录页封面背景设计

## 概述

将登录页从纯紫色渐变改为：`cover.png` 全屏底图 + 紫色半透明渐变叠加 + 左侧品牌文案 + 右侧毛玻璃登录卡片布局。

## 布局

```
┌──────────────────────────────────────────────┐
│  cover.png (background-size: cover, 全屏)      │
│  ┌────────────────────────────────────────┐  │
│  │ 紫色渐变叠加层 (rgba 半透明)              │  │
│  │  ┌──────────────┐   ┌──────────────┐   │  │
│  │  │ 实习能量站     │   │  glass card  │   │  │
│  │  │ 标语文案       │   │  登录表单     │   │  │
│  │  └──────────────┘   └──────────────┘   │  │
│  └────────────────────────────────────────┘  │
└──────────────────────────────────────────────┘
```

## 层级结构

1. **底层** — `cover.png` 全屏铺满，`background-size: cover`
2. **叠加层** — 原有紫色渐变 `linear-gradient(135deg, #667eea, #764ba2)` 以半透明 `rgba` 形式覆盖，保证底图可见
3. **左侧** — 品牌名 + 标语，白色文字带阴影保证可读性
4. **右侧** — 毛玻璃登录卡片，与项目现有 `.glass-card` 风格一致

## 文件变更

| 操作 | 文件 | 说明 |
|------|------|------|
| 移动 | `cover.png` → `frontend/public/cover.png` | Vite 静态资源，路径 `/cover.png` |
| 修改 | `frontend/src/pages/Login.tsx` | 背景改为多层、布局改为左右对半、卡片改用毛玻璃 |

## 技术细节

- CSS `background-image` 多层：`url(/cover.png), linear-gradient(135deg, rgba(102, 126, 234, 0.85), rgba(118, 75, 162, 0.85))`
- 外层容器 `display: flex`，左右各占 50%
- 右侧垂直居中 + 水平居中放置登录卡片
- 卡片样式：`background: rgba(255,255,255,0.15)` + `backdrop-filter: blur(16px)` + 白色边框
- 左侧文案使用 `text-shadow` 保证在底图上的可读性
- 保留原有表单逻辑、测试账号提示不变
