1. 色彩体系

index.html 使用 amber 暖色调。我们将其延展为更完整的色彩系统，增加冷色锚点以适配数据看板的信息密度。

index.html 原色调

amber-50 → amber-500 / red-500 / amber-100 / sky-50

实习能量站 色彩映射

bg: amber-50→sky-50 渐变 / primary: amber-500 / success: emerald-500 / info: blue-500 / danger: red-500

增加绿/蓝色阶用于数据状态标识，保留 amber 作为品牌主色
