1. 各页面的改动范围

实习生 Dashboard

  • 统计卡片 → glass-card 居中数字
  • 成长卡片 → 自定义渐变进度条

  • 操作按钮改为 amber link 风格

  • Table 包裹在 glass-card 中
  • AI 建议 Tag 改为胶囊样式

2. 动效与微交互（参考 index.html）

卡片 Hover

transform: translateY(-2px) + box-shadow 加深
transition: 0.3s 平滑过渡
← 直接复制 index.html .proj-card:hover

庆祝动效 (FR-006)

连续两周完成率 > 90% 触发
轻量 CSS confetti + 卡片金色边框闪烁
不引入动画库，纯 CSS animation

页面入场

卡片从下向上淡入 (fade-in-up)
各卡片 stagger 50ms 依次出现
@keyframes fadeInUp 纯 CSS

进度条动画

进度条从 0 展开到目标值
transition: width 0.6s ease-out
页面加载时触发一次

3. 不做的

  • 不引入动画库（framer-motion 等）—— 纯 CSS animation 足够
  • 不改造页面路由 —— 保持当前条件渲染模式
  • 不改变数据流 —— API 调用、状态管理逻辑不变
  • 不新增依赖 —— 利用已有的 Tailwind + Ant Design
  • 不做移动端适配 —— spec 明确 Desktop only
